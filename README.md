# 顔を認識して文字入力するトーキングエイド

ブラウザで使用するトーキングエイドです。  
Webカメラで顔を認識し、MediaPipeを使って文字入力します。

## 使い方

- ブラウザでページを開きます。
- カメラへのアクセスを許可します。


## 使用している技術

- JavaScript / HTML / CSS
- Google MediaPipe

このトーキングエイドは Google MediaPipe を利用しています。  
MediaPipe は Apache License 2.0 のもとで提供されています。
